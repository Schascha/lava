![Build](https://github.com/Schascha/lava/workflows/Build/badge.svg)

https://schascha.github.io/lava/

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm test
```
